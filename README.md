## 🏠 The Gap in Smart Home Automation

Making a smart home truly "smart" is harder than it looks. Currently, we only have two bad options:
1. **Manual Rules:** You have to manually set up endless "If-This-Then-That" rules, which is exhausting.
2. **Clumsy AI:** Traditional Machine Learning can guess high-level activities (like "Cooking"), but it fails to map them to specific, practical device actions (like "Turn on the stove light and start the exhaust fan").

## 💡 Our Solution: Learn by Watching

We introduce a framework that completely removes the need for manual rule setup and tedious data labeling. 

* **Passive Labeling:** The system quietly runs in the background, learning your habits simply by observing your natural manual actions (e.g., flipping a switch) and ambient sensor data. 
* **Context-Aware Prediction:** We designed a contextual fusion algorithm that looks at your historical data to predict exactly *which device* you want to trigger at a fine-grained level.
